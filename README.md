# Solving Cliff Walking with Reinforcement Learning

## 📌 Project Description

The goal of this project is to replicate a simple scenario defined by Sutton and Barto in 2018 using the concept of reinforcement learning.

The idea is to write a program that would allow an agent to navigate to the goal while avoiding traps and maximizing reward through a process of trial and error.

## ⚙️ Main steps
This project follows a series of logical steps:
- Importing the necessary libraries
- Setting up the environment: field and number of actions, rewards, policy (set of actions), loops for iterations, etc.
- Deterministic model
- Non-deterministic model
- Machine learning
- Conclusion

---

## 📈 KPIs & Analysis
Key metrics included in the project:

- Q-Value(s,a) for each action (a) and state (s)
- Probability of falling and of not falling
